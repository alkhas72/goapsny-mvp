## Surface: Карточка места (bottom-sheet)

Источник: `DESIGN.md §7` (карточка места) + каркас слоя 2 «Окно 3» ([[goapsny-wireframe-layer2-2026-07-06]]). Первая поверхность, прошедшая `/shape` как тест процесса «текст → карточки → прототип → код».

### Composition

**Recipe:** n/a — кастомный bottom-sheet детали объекта поверх живой карты (готового recipe под карту-деталь в UI Craft нет).
**Why:** brief принцип 2 «Приложение — это карта, не мини-сайт»: панель = оверлей поверх карты, тап по пину → сразу карточка, без промежуточного попапа. Принцип 1 (доступность) — пол для всех состояний.

### Content inventory (приоритеты)

- **P0** Иконка категории в ЦВЕТЕ статуса + Название места + крестик ✕ (≥44×44 px)
- **P0** Статус словом + цветом (уровень 1 — НЕ кликабельно)
- **P0** Фото во всю ширину (обязательно по модели → всегда присутствует; с alt-текстом)
- **P0** Адрес
- **P0** Индикатор пандуса — пурпурная точка `#7A5AF8` + текстовая подпись (если есть)
- **P1** «Проверено: [роль] · [когда]»
- **P1** Свёрнутая строка-верификация «▸ Есть на: Яндекс↗ Google↗ Apple↗ OSM↗» (Яндекс первым)
- **P1** «Сообщить об ошибке»
- **P0 (только вид автора заявки)** Янтарная плашка «На проверке у модератора · Видно только вам»

### State lattice

| State | Required / Optional / N-A | Notes |
|-------|--------------------------|-------|
| idle | **Required** | публичное место, данные есть |
| loading | **Required** | скелет по форме карточки; задержка 200мс до показа |
| error | **Required** | причина + повтор («не удалось загрузить место») |
| partial | **Required** | упал блок «Есть на…» или фото → показываем остальное |
| **pending / модерация** | **Required** | вид автора своей заявки (заявка≠объект): янтарная плашка «На проверке у модератора · видно только вам», статус серый. **Единственное состояние, не показанное на каркасе слоя 2 — достраивается.** |
| success | **Required** | подтверждение после «Сообщить об ошибке» (тост/строка) |
| offline | Optional — кэш места + очередь «Сообщить об ошибке» | |
| empty | N-A | фото и статус обязательны → пустого места не бывает |
| conflict | N-A | ур.1 не редактирует на этой поверхности |

См. `references/state-design.md` для контрактов состояний.
